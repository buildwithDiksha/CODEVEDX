## 📌 About

This portfolio serves as my digital identity as a Computer Science Engineering student. It highlights my technical skills, featured projects, and provides a way for recruiters, collaborators, and fellow developers to connect with me.

---

## ✨ Features

- **Responsive Design** — Works seamlessly across all screen sizes (mobile, tablet, desktop)
- **Smooth Navigation** — Single-page layout with clean anchor-based section links
- **Resume Download** — Direct PDF download of my latest resume
- **Project Showcase** — Links to live demos and source code
- **Contact Section** — Easy access to email and GitHub profile

## 🧑‍💻 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/buildwithDiksha/CODEVEDX.git

# Navigate into the project folder
cd CODEVEDX

# Open in browser
open index.html
# or just double-click index.html
```

No dependencies or build tools required — it's plain HTML/CSS/JS.
